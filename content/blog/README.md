# مدونة اللمسة الذهبية — MDX

ضعي ملفات `.mdx` هنا للنشر عبر CMS مستقبلاً.

## مثال frontmatter

```yaml
---
slug: my-article
title: عنوان المقال
excerpt: وصف قصير
metaDescription: وصف SEO
keywords: ["كلمة1", "كلمة2"]
category: hair
publishedAt: 2026-06-01
image: https://images.pexels.com/...
imageAlt: وصف الصورة
featured: false
---
```

المحتوى أسفل الـ frontmatter يُعرض عند تفعيل عارض MDX.
